# 🎮 Simon Say Game

A classic **Simon Say Memory Game** built using **HTML, CSS, and JavaScript**. The game challenges players to memorize and repeat an increasingly long sequence of colors. With each correct round, the difficulty increases by adding a new color to the sequence.

1. Clone this repository

```bash
git clone https://github.com/shivraj2328/Simon-Say-Game.git
```

2. Open the project folder

```bash
cd Simon-Say-Game
```

3. Open `index.html` in your browser.

---

## 🎮 How to Play

1. Press any key to start the game.
2. Watch the highlighted color carefully.
3. Click the colors in the same order.
4. Each new level adds one more color.
5. The game ends if you click the wrong color.
6. Press any key to play again.
