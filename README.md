# Simon Say Game
A simple Simon Say game built using HTML, CSS and JavaScript.

## Features
- Random color sequence generation
- Keyboard to start the game
- Level progression
- Score tracking
- Game over and restart functionality

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Project Structure
```
Simon-Say-Game/
│── index.html
│── style.css
│── script.js
```

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
